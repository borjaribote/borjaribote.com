Web-garatzailea

Nire buruz

Borja Ribote naiz, 28 urte ditut eta Londresen bizi naiz 2017/10/12tik hona.

Londresera etortzeko erabakia ez zen lanagatik izan, ezta nire herrialdeko krisiagatik ere. Izan ere, enpresa handi bateko garatzailea nintzelarik, nire lana utzi nuen Londresera etortzeko. Bidaiatzeko eta guztiengandik urrun zegoen beste hiri batean bizitzeko erabakia ez zuen beharrizanak bultzatu, ilusioak baizik.

Nire izaera humorean oinarrituta eraikita dago, pertsona alai eta baikortzat dut nire burua. Lanean, gainerakoei energia ona helarazten saiatzen naiz; jendeari laguntzen eta animatzen gustatzen zait pertsona batentzako oinarrizkotzat jotzen baitut. Lantaldean lan egiterako orduan, elkartuta egotea oso garrantzitsua dela uste dut, horrela, bata bestean sostengatuko eta animatuko ginatekeelako, eta helburu handiagoak lortuko genituzkeelako.

Akademikoki, ez ditut emaitza hoberenak lortu eta ez dut unibertsitateko ikaskuntzarik, baina hori horrela gertatu da garatzailea izateko nire pasioa eta erabakia berandu eta era latzean heldu zelako. Nire bizitza aldatuz, ezagutzen ez nuen zerbait ikastea erabaki nuen, jakin-mina sortzen zidalako, eta hortxe bertan nire bokazioaz ohartu nintzen. Ikastaroa aurrera ateratzea asko kostatu zitzaidan, besteekin alderatuz, nik ez nuen aurrezagutzarik, baina esfortzua bereizten nauen gauza bat da, eta, hori esker, ikastaroa gainditzearekin batera, aldez aurretik nik baino ezagutza handiago duten pertsonen gainetik lanean hasi nintzen enpresa batean.
Asko ikasi beharra daukat, baina hori oso gustukoa dut, asko disfrutatzen dudalako ikasten eta ikasitakoa erabiltzen. Beraz, prestakuntza jasotzen eta webguneen garatzaile lanean hazten jarraitzea nahiko nuke.

Amaitzeko, gogoeta batekin bukatu nahiko nuke, egunero laguntzen didan esaldi batekin, zu orain hau irakurtzen ari zarenaren erantzulea dena: Ni ameslari bat naiz, ahal duen pertsona bat eta batzuek zoroa naizela uste dutena; baina mundua aldatu ahal dutela pentsatzen duten eta nahiko zoratuta dauden pertsonak soilik dira aldatzen dutenak. Beraz, pentsatu ezberdin.

Nire azken proiektua

Porfolio hau nire azken lana da. Hemen, Interneten dudan leku txiki honetan, erakusten ditut burututako ideiak, lan eta ikasketa orduetan bihurtutako ideiak, lan on bat gauzatzeko ilusio, jakin-min eta konfiantzan oinarritutakoak.

Emaitza ezberdinak lortzeko bilaketa horretan, gauzak hausnartzea gustuko dut, hau da, mundu guztiak berdin egiten duen era horretan burutu behar direla ezartzen duten arauengandik aldenduz. Horrenbestez, ez zait gustatzen ezer zoriaren esku ustea, adibidez, kolore bakoitza, itxura bakoitza, kokapena eta abar. Zentzua eta azalpena du, eta horrek eraman nau UX/UI, tipografia o koloreen psikologia aztertzera eta ikastera.

Sarritan erabilitako bidea Internet izan da. Online ikastaroek eta web orriek laguntza handia eskaini didate proiektu hauek burutzeko. Horiei esker eskuratu ditut Photoshop eta JavaScript tresna informatikoei buruz dakidan guztia, eta, gainera, HTML eta CSS arloak hobetu ditut, gehienbat.

Orrialdea optimizatzeko, era erraz, intuitivo eta moldagarri batean gauzatu da, hots, orrialdea edozein pantailara egokitzeko flexbox erabili da, bilaketa erraz egiteko web kokapen zuzena eta web kokapen zuzena eta erabiltzale-interfaza arrunt bat erabiliz.

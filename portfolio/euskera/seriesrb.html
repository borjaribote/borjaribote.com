Web-Orria

Web-orri hau etorkizunean zertan lan egin nahi nuen erabakitzeko arrazoia da. Hemen hasi zen nire barnean iraultza, web-orri honi esker web-diseinua aurkitu bainuen. Hasi nintzenean, ez nekien ezer HTML eta CSS tresna informatikoei buruz, baina, hasi eta gero, jakin-mina sortu zitzaidan, ideiak bakarrik zentozen eta mundu berri bat begiztatzen hasten den jaioberri baten antzera sentitu nintzen.

Nire bizitzan mugarria izan den web-orri hau, egia esan, ikasgelarako lan bat izan zen, eta arau zehatz batzuk bete behar nituen. Hasteko HTML5 eta CSS3, tresna informatikoak soilik erabil zitezkeen, baina horrekin nahiko ez zitzaidanez, nire kabuz aztertzen hasi nintzen javascript kode txiki bat gehitu nion, nire ustez, nabigatzeko erraztasuna ematen zuena eta betekizunetan eragin handirik ez zuena.

Ikasgelakoei asko gustatu zitzaien eta emaitza altua izan zen, nahiz eta garai hartan emaitzak garrantzi handirik ez izan, nire benetako bokazioa aurkitu bainuen.

Mapa eta Menua

Web-orriaren hasieran saileko pertsonai guztiak biltzen dituen argazki bat nagusi da, eta, era berean, mapa bat da; bertan sakatuz gero, bakoitzak orrian duen bere gunera eramango zaitu.
   Bestetik, menua dugu, eta, berari esker, edozein lekura mugituko gara, edozein lekutan egonik ere.

Koadroa

Hau, “modal” koadro estatuko bat da, eta orrialde nagusian dago.

Bere barnean, saileko irudi batzuen bitartez nabiga dezakegu, ‘hurrengoa’ edo ‘aurrekoa’ hitzen gainean sakatuz edota banan-banan txandako salto eginez.

Igo

Amaieran, gora begira dagoen ikono bat aurkituko dugu. Gainean sakatuz gero, orrialdeak gora egingo du automatikoki eta irudia era barregarrian aldatzen da.

Ikono honen funtzioa Javascript tresna informatikoaren bitartez eginda dago, eta, era berean, orrialdearen kode bakarra da.

Hauteman dezakezunez, orrialdea HTML eta CSS teknika ezberdinez osatuta dago. Eta “Ez da berdin ikusi eta begiratu, ezta entzun eta aditu ere”, esaldiari jarraiki, hemen duzu orrialdearen lotura, bertan barrena ibiltzeko eta hobeto ezagutzeko.
